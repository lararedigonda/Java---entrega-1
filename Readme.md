Proyecto Final - Simulador de Inscripción a Talleres de Cerámica

Descripción

Este proyecto consiste en un simulador de inscripción a talleres de cerámica. 
Permite registrarse en distintas actividades, gestionar inscripciones y visualizar la información en tiempo real.

Lógica del simulador

a. Se completa el formulario
b. Se validan los datos ingresados
c. Se verifica:
   - edad 
   - duplicación de inscripcion
d. Se guarda la inscripción
e. Se actualiza la vista en el HTML
f. Se muestra confirmación
